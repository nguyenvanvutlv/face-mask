## 🚀&nbsp; Installation

1. clone repo
```
https://github.com/nguyenvanvutlv/face-mask.git
```
2. Di chuyển đến thư mục chính

```
cd face-mask
```
3. Cài thư viện

```
pip3 install -r requirements.txt
```


## :trophy: Run

```
python main.py
